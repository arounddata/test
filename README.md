# KIMPL1 — Вычисление замыкания системы функциональных зависимостей

**Текущая версия:** 11.18

---

## Введение

Программа вычисляет замыкание системы функциональных зависимостей (ФЗ) с использованием правил транзитивности и псевдотранзитивности.

Алгоритм основан на оригинальном PL/I коде (1986) и адаптирован для веб-интерфейса.

---

## Установка и запуск

1. Скопируйте все файлы проекта в одну папку:
   - `index.html`
   - `style.css`
   - `script.js`
   - `README.md`

2. Откройте `index.html` в любом современном браузере (Chrome, Edge, Firefox).

Никаких дополнительных зависимостей не требуется — все библиотеки подключаются через CDN.

---

## Формат ввода

ФЗ вводятся в левой панели в **составной форме** (с-форма):

- `A-B` — атрибут A определяет B
- `A-B-C` — атрибут A определяет B и C (две ФЗ: A-B, A-C)
- `A*B-C` — атрибуты A и B совместно определяют C
- `A*B-C-D` — атрибуты A и B совместно определяют C и D

**Разрешённые символы в именах атрибутов:** любые, кроме `*`, `-` и пробела.
