Краткая спецификация финальной версии «11.13»:
Пункт	Описание
Название	Вывод замыкания системы функциональных зависимостей
Версия	11.13 (отображается в правой части информационной панели)
Формат ввода	С-форма (строковые атрибуты), например A-B, A*B-C
Панели	Левая: исходные данные (редактируемая таблица), Центральная: исходная система ФЗ (после проверки), Правая: замыкание
Преобразования	С-форма → Ч-форма (числовая) → Б-форма (битовая) для алгоритма, затем обратно
Алгоритм	Замыкание по правилам транзитивности и псевдотранзитивности (оригинальный PL/I код)
Кнопки	Открыть, Проверить, Рассчитать, Сохранить
Горячие клавиши	Ctrl+O, Ctrl+R, Ctrl+Shift+S
Сохранение	XML-файл в с-форме (без битовых комментариев)
Очистка при загрузке	При открытии файла очищаются все панели
Очистка при добавлении	При нажатии «+» очищаются центральная и правая панели
Удаление <fdsc>	При загрузке файла удаляется старая секция <fdsc>
Стиль	Три колонки, высота строк 38px, заголовки панелей с количеством ФЗ
