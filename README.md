## Вычисление замыкания системы функциональных зависимостей

## Описание

Программа вычисляет замыкание системы функциональных зависимостей (ФЗ) с использованием правил транзитивности и псевдотранзитивности.

## Формат ввода

ФЗ вводятся в левой панели в **с-форме** (строковые атрибуты):

- `A-B` — атрибут A определяет B
- `A*B-C` — атрибуты A и B совместно определяют C

## Действия по Открыть (до нажатия «Проверить»)
При загрузке XML-файла (кнопка Открыть) выполняются следующие действия и проверки:

1. Чтение файла
Проверяется, что файл успешно прочитан (FileReader)

2. Удаление старых результатов
Удаляются все блоки <fdsc>...</fdsc>

Удаляются комментарии <!-- FDS Closure -->

Удаляются лишние пустые строки

3. Поиск тегов <fd...>
Ищутся все теги вида <fd...> (с любым содержимым после fd)

Из каждого тега извлекается текст

Пустые теги игнорируются

4. Проверка наличия ФЗ
Если не найдено ни одного тега с ФЗ → ошибка "Не найдено ни одной ФЗ"

5. Заполнение appState
originalFds заполняется загруженными строками

Очищаются:

isDataValid = false

numericFds = null

attrMap = null

attrMapReverse = null

numericN = null

closureCform = null

closureResult = null

6. Сброс панелей
Очищается центральная панель (показывается "Нажмите «Проверить» после ввода данных")

Очищается правая панель (показывается "Нет результатов")

Сбрасывается количество атрибутов в информационной панели

## Формат файла

XML-файл должен содержать теги `<fd...>`:

```xml
<fdsi>
    <fd1>A-B</fd1>
    <fd2>A*B-C</fd2>
</fdsi>
